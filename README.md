DataStructureInJS
=================

Implementations of some simple data structure in JavaScript

- Queue
- Stack
- Linked List
- Double Linked List
- Graph
  - Adjacency list representation
  - Breadth-first Search
  - Depth-first Search
  - Dijkstra's shortest-path algorithm
- Binary Heap
  - Max-Heap
  - Min-Heap
- Priority Queue
- BST
